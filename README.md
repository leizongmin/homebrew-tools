# homebrew-tools

Personal Homebrew tap

## 安装 luckypg

```bash
brew tap leizongmin/tools
brew install luckypg
```

## 开发

执行以下命令测试 `luckypg.rb` 安装脚本：

```bash
brew install --verbose --debug --build-from-source luckypg.rb
```
